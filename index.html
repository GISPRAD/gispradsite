Projetos

Aqui ficam os projetos desenvolvidos no laboratório. Cada projeto aparece como um cartão com uma breve explicação.

Projeto A — Monitoramento e diagnóstico espacial

Desenvolvimento de rotinas para análise temporal e espacial de variáveis ambientais, integrando dados de campo, sensores e produtos de satélite para apoiar decisões.

Projeto B — WebGIS e visualização interativa

Construção de mapas e aplicações web para disponibilizar resultados de pesquisa, camadas temáticas e painéis de indicadores de forma acessível.

Projeto C — Automação de processamento (pipelines)

Padronização e automação de fluxos de dados (ETL), garantindo reprodutibilidade, versionamento e relatórios automáticos.
